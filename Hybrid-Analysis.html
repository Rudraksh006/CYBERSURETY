Hybrid Analysis is a free malware analysis service that detects and analyses unknown threats using a unique Hybrid Analysis technology.

Analysing Files Using Hybrid Analysis:

The first step is to go to the website of Hybrid Analysis. 

Click on the “Drag & Drop For Instant Analysis” button & then select the file which you want to analyse.  

After selecting the file, it will ask you to enter some details.

Entering your email address & comment is optional. 

Tick the “Do not submit my sample to unaffiliated third parties” condition & also tick the “I consent to the Terms & Conditions and Data Protection Policy” condition but read the Terms & Conditions as well as the Data Protection Policy before ticking the “I consent to the Terms & Conditions and Data Protection Policy” condition.

Untick the “Allow community members to access sample” condition.   

Verify the “I’m not a robot” captcha.   

After doing the above changes, click on the “Analyze” button.  

Now the file will be uploaded on the server.  

Once the file has been uploaded, it will start analysing the file.  

Once the file has been analysed it will show the result, if the file was found safe then it will show “no specific threat”, and if the file wasn’t found safe then it will show “suspicious”.
